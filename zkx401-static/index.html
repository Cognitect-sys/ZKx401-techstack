Quick Start

Get started with ZKx401 in minutes. Our npm package provides everything you need to implement privacy-enhanced transactions on Solana.

Installation 
npm install @payai/x402-solana
Basic Usage 
import { ZKx401 } from '@payai/x402-solana';

const zkx401 = new ZKx401({
  network: 'mainnet-beta',
  privacyLevel: 'high'
});

// Create anonymous transaction
const tx = await zkx401.createAnonymousTransaction({
  amount: 1000000, // 1 SOL
  recipient: 'recipientAddress',
  private: true
});
